## 1. Pemrograman Jaringan
Pemrograman jaringan merupakan bidang dalam ilmu komputer yang berfokus pada pengembangan aplikasi yang mampu melakukan komunikasi dan pertukaran data antar perangkat melalui jaringan komputer. Proses komunikasi ini dapat terjadi pada **LAN (Local Area Network)** maupun **WAN (Wide Area Network)** sehingga memungkinkan sistem bekerja secara terdistribusi.

---

## 2. Konsep Dasar Pemrograman Jaringan
Beberapa konsep utama dalam pemrograman jaringan meliputi:

- **Arsitektur Client–Server**  
  Klien mengirimkan permintaan, server memproses dan memberikan respon.

- **Protokol Jaringan**  
  Aturan komunikasi data seperti HTTP dan SMTP.

- **Socket**  
  Mekanisme komunikasi jaringan menggunakan TCP atau UDP.

- **Sinkron & Asinkron**  
  Sinkron menunggu respon, sedangkan asinkron memungkinkan proses lain tetap berjalan.

- **Pemrograman Web**  
  Aplikasi dijalankan di server dan diakses melalui browser.

- **Pemrograman Mobile**  
  Umumnya menggunakan REST API untuk komunikasi dengan server.

---

## 3. Peran Node.js dalam Pemrograman Jaringan
Node.js merupakan **runtime JavaScript di sisi server** yang menerapkan konsep **event-driven** dan **asynchronous programming**. Dengan pendekatan ini, Node.js mampu:
- Menangani banyak koneksi secara bersamaan
- Mengelola operasi input/output secara efisien
- Menggunakan **event loop** untuk mencegah proses blocking

---

## 4. Keunggulan Node.js
Keunggulan utama Node.js antara lain:
- **Skalabilitas Tinggi**
- **Responsivitas Baik**
- **Cross-Platform**
- **Satu Bahasa untuk Client & Server**
- **Ekosistem NPM yang Luas**
